# Idiomatic code test

## Installation

Use yarn or npm to install all packages dependencies

```bash
yarn
```

or

```bash
npm install
```

## Run the project

```bash
yarn dev
```

or

```bash
npm run dev
```

Then you can open the [localhost:3000](http://localhost:3000)
